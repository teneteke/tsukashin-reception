# つかしん窓口精算

テニスラウンジつかしんの受付業務用 PWA（Progressive Web App）です。

**公開URL:** https://teneteke.github.io/tsukashin-reception/

## 概要

- 当日の来館者一覧管理（予約取り込み・ウォークイン追加）
- 会計明細の記録（商品追加・数量変更・決済方法選択）
- 領収書の発行（jsPDF による A4 PDF 生成）
- CSV インポート/エクスポート（会員マスタ・決済方法・予約リスト）

## 推奨環境

- **ブラウザ:** Google Chrome / Microsoft Edge（PC）
- **画面:** 1024px 以上の横幅（横向き想定）
- Safari / Firefox は OPFS サポートが限定的なため非推奨

## データの保存場所とセキュリティ

| 項目 | 内容 |
|------|------|
| **保存場所** | 全データはブラウザの Origin Private File System（OPFS）内に SQLite として保存されます。**サーバーへのデータ送信は一切ありません。** |
| **通信** | アプリは完全オフラインで動作します。GitHub Pages はアプリのコード（HTML/CSS/JS）を配信するだけで、業務データには一切アクセスしません。 |
| **個人情報** | 会員名・電話番号・会計データ等はすべて各 PC のブラウザ内にのみ存在します。別の PC や別のブラウザからは参照できません。 |
| **アクセス制御** | アプリ自体に認証機能はありません。URL を知っていれば誰でもアプリ画面を開けますが、データは各ブラウザに閉じているため、他者のデータは見えません。 |
| **バックアップ** | ブラウザのサイトデータを削除すると全データが失われます。日次で CSV エクスポートまたは SQLite エクスポートを行い、バックアップを保管してください。 |

## 使い方

1. 上記 URL をブラウザで開く
2. アドレスバー右端のインストールアイコンから PWA としてインストール可能
3. 画面 1（来館者一覧）の「更新」ボタンから CSV をインポート
4. 各来館者の行で商品追加・出席確認・会計処理

## 技術スタック

- Vanilla JavaScript（フレームワークなし）
- sql.js（SQLite in WebAssembly）+ OPFS 永続化
- jsPDF + Noto Sans JP（日本語 PDF 生成）
- Service Worker（オフライン対応）
- GitHub Pages（ホスティング）
